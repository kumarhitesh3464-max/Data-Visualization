## 📌 Project Overview

This project was completed as part of the **CodeAlpha Data Analytics Internship**.

The objective is to transform raw employee data into meaningful visualizations using Python.

---

## 🎯 Objectives

* Visualize data distributions
* Compare categories
* Analyze relationships
* Present insights through graphs

---

## 🛠️ Tools & Libraries

* Python
* Pandas
* Matplotlib
* Seaborn

---

## 📂 Dataset

Employee Dataset

Columns:

* EmployeeID
* Name
* Department
* Age
* Salary

---

## 📋 Visualizations Created

### 1️⃣ Histogram

Shows the distribution of employee ages.

### 2️⃣ Scatter Plot

Displays the relationship between Age and Salary.

### 3️⃣ Bar Chart

Compares the average salary across departments.

---

## 📈 Key Insights

* Most employees belong to the middle-age group.
* Salary tends to increase slightly with age.
* The IT department has the highest average salary.
